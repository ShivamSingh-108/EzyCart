# 🛒 EzyCart - Full Stack E-commerce Application

EzyCart is a full-stack e-commerce web application designed to provide a seamless online shopping experience. It includes user authentication, product management, and real-time features.

## 🚀 Features
- User authentication (login/signup)
- Product listing and management
- Add to cart functionality
- Real-time updates using sockets
- Responsive user interface

## 🛠️ Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Real-time: Socket.io

## ⚙️ Setup Instructions
1. Clone the repository
2. Install dependencies using `npm install`
3. Configure environment variables
4. Run backend and frontend servers

## 📌 Note
This project was explored and customized to understand full-stack development, API integration, and real-time communication.
